Chromebook:

Go to settings app and find "Linux Development Enviroment" install that, now follow the steps for linux, as the steps are the same.

Linux:

1: Go to terminal
2: Copy the command:
sudo apt update
3: Skip step 3, 4, and 5 if you already have c++ installed, but if don't have c++ installed, then copy this command:
sudo apt install g++
4: When downloading c++, it'll ask you if you want to install c++ or not, type "y" to install
5: Now wait until c++ is installed on the terminal
6: Now that you have c++ installed, Download ladder climb
7: FOR CHROMEBOOK ONLY: drag "ladder_climb.cpp" to linux files in files app
8: FOR LINUX ONLY: Make sure the file path is "/home/[your username]/ladder_climb.cpp" replace [your username] with your username
9: On terminal copy the two LAST COMMANDS:
c++ ladder_climb.cpp
After you did it, type in the last command:
./a.out
And there you go!
